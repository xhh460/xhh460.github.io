Ubuntu Linux shell命令

1.认识shell命令

  • shell命令的一般格式

  • command [-option] parameter1 parameter2...

例如：

返回上级目录：

返回登陆用户的主目录：

  • ls——显示指定目录的清单

    一般情况下，我都是用ls -l命令来查看，因为这个命令可以输出文件或目录的详细信息。

  • cat——显示文件的内容

    一般用cat -n比较好一点，因为这样可以显示行号

    比如我要查看test1.sh这个文件，有了行号会更加清晰更具可读性。

  • more——逐渐显示文件中的内容

  • file——显示文件或目录的类型

3.Linux文件查找类命令

  • whereis——查找文件位置

此方法与locate相似。

  • locate——查找绝对路径中包含指定字符串的文件

4.Linux文件及目录的创建

  • touch——生成文件或修改文件的存取时间

  • mkdir——创建新目录

5.Linux文件及目录的删除

  • rmdir——删除目录

  • rm——删除文件或目录

    如果目录删除不了的话，一方面是权限的问题，另一方面可以加参数-f强制删除。

6.Linux文件及目录的复制

  • cp——文件及目录的复制

    -r 递归复制目录，一般可能会用到-r

7.Linux文件及目录的移动

  • mv——移动文件位置

    -f 不论目标文件或目录是否存在，均不提示。

8.Linux文件及目录的归档，打包

  • tar
