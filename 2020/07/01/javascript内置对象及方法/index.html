如果不局限与字母顺序排序,就需要给sort方法提供函数，函数内传两个参数a和b。

  • 若 a 小于 b，在排序后的数组中 a 应该出现在 b 之前，则返回一个小于 0 的值。
  • 若 a 等于 b，则返回 0。
  • 若 a 大于 b，则返回一个大于 0 的值。

如果比较的两值刚好是数字,a-b升序,b-a降序。
